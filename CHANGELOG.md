# Change Log

All notable changes to the "vscode-bend" extension will be documented in this file.

## [0.1.0] - Initial Release

- **Semantic Highlighting**: Added comprehensive TextMate grammar for Bend types, keywords, operators, and control structures.
- **Language Server**: Implemented a standalone Node.js Language Server backend.
- **Diagnostics**: Real-time syntax validation powered by `bend check`.
- **Hover & Outline**: Hover documentation and Document Symbols outline support.
- **CodeLens Execution**: Direct "Run this file" buttons appearing over `def main` functions.
- **Environment Detection**: Status Bar indicator showing active compiler path and default runtime (`c`, `rs`, or `cu`).
